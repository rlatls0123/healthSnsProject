## ✨ 소개

운동 기록 앱과 식단 관리 앱이 분리되어 있어 발생하는 불편함에 주목했습니다. 두 애플리케이션을 병행하는 사용자들이 겪는
번거로움을 해소하고자 하나의 플랫폼에서 운동 기록과 식단 관리를 동시에 관리할 수 있는 통합 애플리케이션을 구축했습니다.

📖 개요
개발 기간: 2024.04 ~ 2024.06



## 🛠 주요 기능
- **통합 기록 관리 (CRUD)**: Java와 Android Studio를 활용하여 운동 루틴과 식단 데이터를 한눈에 확인하고 자유롭게 생성, 수정, 삭제할 수 있는 직관적인 인터페이스를 제공합니다.
- **Firebase 기반 인증 시스템**: Firebase Authentication을 연동하여 이메일 형식의 안전한 회원가입 및 로그인 기능을 구현했습니다. 이메일 식별자를 통해 향후 비밀번호 재설정 등 사용자 계정 복구의 확장성을 확보했습니다.
- **데이터 무결성 검증**: 입력 단계에서 실시간 유효성 검사(Validation) 로직을 적용하여 공백이나 잘못된 형식의 데이터가 저장되는 것을 방지하고 시스템의 안정성을 높였습니다.
