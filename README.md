## ✨ 소개

운동 기록 앱과 식단 관리 앱이 분리되어 있어 발생하는 불편함에 주목했습니다. 두 애플리케이션을 병행하는 사용자들이 겪는
번거로움을 해소하고자 하나의 플랫폼에서 운동 기록과 식단 관리를 동시에 관리할 수 있는 통합 애플리게이션을 구축했습니다.

📖 개요
